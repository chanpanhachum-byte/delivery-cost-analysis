# Delivery Cost and Failed Delivery Analysis

## Project Overview
This is a self-initiated data analysis project using a public dataset to evaluate delivery performance and cost efficiency.

## Objectives
- Identify which delivery provider has the highest number of failed deliveries
- Analyze the cost associated with failed deliveries
- Evaluate performance using KPIs

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Conditional Formatting

## Key Insights
- Identified provider with highest failure rate
- Measured cost impact of failed deliveries
- Built KPI dashboard

## Files
- delivery_analysis.xlsx

## Note
This project uses a publicly available dataset for learning purposes.
